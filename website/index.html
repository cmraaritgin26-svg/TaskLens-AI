AI-supported review

Coach insights across the whole app

The AI Coach can review task completions, deadlines, nutrition, vitals, water, symptoms, mood, ketosis, and journal patterns together to suggest practical next steps.

The app does not diagnose medical conditions. It can flag patterns and suggest when urgent help may be appropriate.

Task trend Late deadlines are checked against mood, symptoms, and journal notes.
Health trend Vitals and nutrition are reviewed with recent symptoms and hydration.
Journal safety Concerning language can trigger supportive guidance and crisis resources.

Privacy and safety

Built for personal tracking

Journal entries stay private inside the app unless the user opens them. AI review looks for patterns and shows concerns to the user only.

Cloud AI features are optional and only work after the user enables internet sharing and saves an HTTPS AI backend URL.

AI features require the configured backend service and an active OpenAI API key with available quota.
